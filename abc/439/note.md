# D 問題
まずは、j が最大の場合について調べる。

3と7の倍数の登場数を記録する。
count_three においては、key を3の乗数とする。例えば、3が一つ、6が二つなら {1:2, 2:2}となる。count_sevenにおいても同様。
```cpp
map<int, int> count_three, count_seven
```

前から順に約数を調べ、それぞれに記録していく。3 か 7 なら、その商を count の map に記録し、５ならその商 における登場数を count で調べ、その組み合わせを計上する。ついでに、j が最小の場合のために A を vector に保存しておく。

複数の約数を持っていることもあるので、else ではなく全て if で調べる。

同様のことを、逆順にもう一度実行して、j が最小の場合について調べる。
