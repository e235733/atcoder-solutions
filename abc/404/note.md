# C 問題
一つの点から辿って、またその点に戻って来れ、全ての点を探索できたかを検査する。

無向グラフを作成する。(index と n を対応させる) 
```cpp:g
vector<vector<int>> g(n+1);
```

全ての点から出る辺が 2 本であることを調べ、そうでなければ即終了。

点 1 から順に調べていき、逐次訪問済みかどうかを記録していく。元に戻ってきたら終了し、全ての点が訪問済みならサイクルグラフ。
```cpp:is_visited
vector<bool> is_visited(n+1, false);
```

# D 問題
ナップサック問題的に求める。

動物の数を1つずつ増やしていき、それぞれを2回以上見る方法を調べる。key は金額、val は 見られる動物の種類とそれらの数。index が動物の番号に対応している。
```cpp:dp
map<ll, vector<int>> dp = {(0, vector<int>(n, 0))};
```

i 番目の動物について、dp の全ての要素について、val の i 番目 の値が 2 未満になっていないか調べる。

２未満だった場合、それを解消する全ての選び方をして、dp に insert し、それ自身は削除する。
