# D 問題
それぞれの累積和を sum_a, sum_b, sum_c とすると、
最大らしさは以下の式で計算できる
```math
sum_a[x] + sum_b[y] - sum_b[x] + sum_c[n] - sum_c[y]
```

この式において、$sum_c[n]$ は一定である。$sum_a - sum_b$ を $d$ と、$sum_b - sum_c$ とすると、以下の式が最大になる場合を求めれば良い。

```math
d[x] + e[y]
```

全探索すると O(N^2) になるので、少し工夫する。x 以下で最大の d とそのときの x を示す max_d を作成し、以下の式を後ろから探索し最大値を求める。
```math
max_d[y-1] + e[y]
```

その後、求めた x, y をらしさを計算する式に代入し、解答する。
