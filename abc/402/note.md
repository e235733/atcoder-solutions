# C 問題

それぞれの料理が何日後に食べられるようになるのかを調べ、i 日後に食べられるようになる料理の数を求める。そして、それらの累積和が答えになる。

## i 日後に食べられるようになる料理の数

料理に含まれる食材、食材ごとの食べられるようになる日、 vector を用意する
```cpp:ingredients
vector<vector<int>> ingredients(m);
```
```cpp:date
vector<int> date(m+1);
```

i 日目における食べられる料理の数を記録する
```cpp:num_edible
vector<int> num_edible(n+1);
```

それぞれの料理における、a の最大値を求める。それらを max_a とし、num_edible[max_a]++ する。

## 累積和を求める

num_edible の i=2 以降に対し、直前の要素に現在の要素を足し合わせていく。
num_edible[i] += num_edible[i-1]

これが答えになっているので出力

# D 問題

2点の和を n で割った余りが等しければ平行になっていそう。

## 余りが等しいものの数を調べる
```cpp:remainder
map<int, int> remainder;
```

## 組み合わせの数を調べる
交わるものは、余りが同じもの同士の組み合わせ数の和になる。同じあまりになるものの数が r とすると、その組み合わせは rC2 であり、これらの和を求める。

そして、これをすべての組み合わせ mC2 から引けば答えになる。
