# C 問題
有向グラフを作成し、深さ優先探索する。

薬品は加えることしかできないので、その操作を辺とする。危険な状態のノード除いて、最初の何もない状態から全ての薬品が混ざった状態が連結なのかを調べる。

各ノードからの辺は、2進数表記したときに 0 である部分を1つだけ 1 に変えたノードへ向かう。

2^(k-1) が、k を n から順に引けるか検査していき、引ければ順次ノードの値から引いていく。引けなければ、その値を足したノードに向かう辺を追加する。

危険な状態のノードを削除したのち、深さ優先探索。

# D 問題
コーラの減り方に着目する。持っている本数以下で交換できる i のうち、減りが最小のものを選び続ければよい。

## 減少量を計算
a と減少量を pair にして vector を作成
```cpp:dec
vector<pair<ll, ll>> dec(m);
```

## 本数ごとに最小の減少量を求める
持っている本数ごとの、最小の減少量を求めて map で管理
```cpp:min_dec
map<ll, ll> min_dec;
```
dec を走査して、a における最小減少量を記録。そして、min_dec を前から走査して、それ以前での最小値にする。
