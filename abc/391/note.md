# D問題

## 列ごとに、ブロック map の vector を作成
```
vector(位置x)<map<(位置y), (ブロックi)>>
```

## 列の中にあるブロックの最小数
```
int min_num
```
これよりブロックが多い列は消え切らない

## 下から i 番目のブロックの高さの最大値
```
vector(下から i)<(最大高さ max_hight)>
```

## 下から i 番目のブロックが消える時刻
```
vector(下から i)<(時刻 rm_t)>
```
消えないブロックの時刻は 10^9

## 各ブロックが消える時刻
```
vector(ブロック i)<(時刻 rm_t)>　
```
